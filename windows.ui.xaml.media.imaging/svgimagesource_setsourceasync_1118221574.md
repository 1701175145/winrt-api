# Windows.UI.Xaml.Media.Imaging.SvgImageSource.SetSourceAsync

## -description
Sets the source image for an **SvgImageSource** by accessing a stream and processing the result asynchronously.

## -parameters
### -param streamSource
The stream source that sets the image source value.

## -returns
An [SvgImageSourceLoadStatus](svgimagesourceloadstatus.md) value that indicates whether the operation was successful. If it failed, indicates the reason for the failure.
