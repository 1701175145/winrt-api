# Windows.UI.ViewManagement.Core.CoreInputViewHidingEventArgs.CancelHideIfAllowed

## -description

Gets or sets whether the application can cancel the [Windows.UI.ViewManagement.Core.CoreInputView.PrimaryViewHiding](coreinputview_primaryviewhiding.md) event. In some cases, such as if the input pane was invoked manually by the user, the keyboard policy can override this ability.

## -property-value

True, if the application can cancel the hiding event. Otherwise, false.
