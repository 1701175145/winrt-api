# Windows.UI.ViewManagement.Core.CoreInputViewShowingEventArgs.CancelShowIfAllowed

## -description

Gets or sets whether the application can cancel the [PrimaryViewShowing](coreinputview_primaryviewshowing.md) event. In some cases, such as if the input pane was invoked manually by the user, the keyboard policy can override this ability.

## -property-value

True, if the application can cancel the [PrimaryViewShowing](coreinputview_primaryviewshowing.md) event. Otherwise, false.
