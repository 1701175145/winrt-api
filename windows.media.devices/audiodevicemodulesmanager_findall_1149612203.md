# Windows.Media.Devices.AudioDeviceModulesManager.FindAll

## -description
Retreives a read-only list of all audio device modules associated with the audio device for which the **AudioDeviceModulesManager** was initialized.

## -parameters

## -returns
A read-only list of all audio device modules
