To get the low-level exception associated with the error, call [TimedMetadataTrackError.Propagate](timedmetadatatrackerror_propagate.md) which will cause the underlying exception to be raised.## -examples## -see-also[TimedMetadataTrackError::Propagate](timedmetadatatrackerror_propagate.md)
