# Windows.UI.Input.Inking.InkPresenterRuler.Width

## -description
Gets or sets the width of the [InkPresenterRuler](inkpresenterruler.md).

## -property-value

The width in device-independent pixel (DIP).

+ Default: 120 device-independent pixel (DIP)
+ Minimum: 120 device-independent pixel (DIP)
