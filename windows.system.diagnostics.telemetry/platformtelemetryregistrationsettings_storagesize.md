# Windows.System.Diagnostics.Telemetry.PlatformTelemetryRegistrationSettings.StorageSize

## -description
Gets the storage size of the Windows telemetry tenant, in bytes.

## -property-value
Storage size for the telemetry tenant, in bytes. Value must be greater than 0 and less than 100 megabytes.
