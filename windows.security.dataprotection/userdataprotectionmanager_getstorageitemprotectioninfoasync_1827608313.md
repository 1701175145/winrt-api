# Windows.Security.DataProtection.UserDataProtectionManager.GetStorageItemProtectionInfoAsync

## -description
Gets the UserDataStorageItemProtectionInfo object of a storage item.

## -parameters
### -param storageItem
The storageItem for which protection info is being queried.

## -returns
When the call to this method completes successfully, it returns a UserDataStorageItemProtectionInfo object concerning the storage item.
