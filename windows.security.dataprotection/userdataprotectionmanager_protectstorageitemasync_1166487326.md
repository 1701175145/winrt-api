# Windows.Security.DataProtection.UserDataProtectionManager.ProtectStorageItemAsync

## -description
Protects a user data storage item.

## -parameters
### -param storageItem
The input, protected storage item.

### -param availability
The availability state.

## -returns
When the call to this method completes successfully, it returns a UserDataStorageItemProtectionInfo object that contains the status of the protected stream.
