# Windows.Security.DataProtection.UserDataProtectionManager.ProtectBufferAsync

## -description
Protect a user data buffer.

## -parameters
### -param unprotectedBuffer
The input, unprotected buffer.

### -param availability
The device's availability state.

## -returns
When the call to this method completes successfully, it returns a UserDataStorageItemProtectionInfo object that contains the status of the protected buffer.
