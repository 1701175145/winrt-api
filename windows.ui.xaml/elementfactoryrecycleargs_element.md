# Windows.UI.Xaml.ElementFactoryRecycleArgs.Element

## -description
Gets or sets the [UIElement](uielement) object to recycle when calling [RecycleElement](ielementfactory_recycleelement_1023702976).

## -property-value
The [UIElement](uielement) object to recycle.

## -remarks
See the remarks for the [DataTemplate](datatemplate)'s [RecycleElement](datatemplate_recycleelement_1023702976) method.
