# Windows.UI.Xaml.EffectiveViewportChangedEventArgs.BringIntoViewDistanceX

## -description
Gets the sum of translation in the X-axis that would be required to bring the [FrameworkElement](frameworkelement) into view of each viewport containing the element.

## -property-value
The translation in the X-axis that would be required to bring the [FrameworkElement](frameworkelement) into view of each viewport containing the element

## -remarks
The value does not account for the possibility of a scrolling control that disabled scrolling.
