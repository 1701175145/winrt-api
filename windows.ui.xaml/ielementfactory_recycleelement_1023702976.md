# Windows.UI.Xaml.IElementFactory.RecycleElement

## -description
Recycles an [UIElement](../windows.ui.xaml/uielement) that was previously retrieved using [GetElement](ielementfactory_getelement_92222689).

## -parameters
### -param args
An instance of [ElementFactoryRecycleArgs](elementfactoryrecycleargs).
