# Windows.Networking.NetworkOperators.MobileBroadbandDeviceInformation.FirmwareInformation

## -description
Gets the device-specific firmware information for the mobile broadband network device.

> [!NOTE]
> This functionality is only available to mobile operator apps and Windows Store app given privileged access by mobile network operators.

For permission to use this API, please reach out to [Network Operators API Permissions](mailto:netopperm@microsoft.com).

For technical help with this API, please reach out to [Network Operators API Help](mailto:netophelp@microsoft.com)

## -property-value
The device-specific firmware information for the mobile broadband network device.
