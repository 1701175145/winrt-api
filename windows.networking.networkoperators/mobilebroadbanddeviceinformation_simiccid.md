# Windows.Networking.NetworkOperators.MobileBroadbandDeviceInformation.SimIccId

## -description
Gets the SIM integrated circuit card identifier (ICCID) for the mobile broadband network device.

## -property-value
The SIM integrated circuit card identifier (ICCID) for the mobile broadband network device.

## -remarks
You can use the [SimIccId](mobilebroadbanddeviceinformation_simiccid.md) value to identify and provision the mobile broadband network device.
