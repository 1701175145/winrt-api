# Windows.Security.Authentication.Web.Provider.WebAccountManager.AddWebAccountForUserAsync

## -description
Adds a new web account for a specific user (for use on multi user aware apps).

## -parameters

### -param user
The Windows user for which an account must be authenticated.

### -param webAccountId
The Id for the web account.

### -param webAccountUserName
The user name for the web account.

### -param props
Properties for the web account.

### -param scope
The scope of the web account.

### -param perUserWebAccountId
The per-user web account Id.

## -returns
When this method completes, it returns the new web account.
