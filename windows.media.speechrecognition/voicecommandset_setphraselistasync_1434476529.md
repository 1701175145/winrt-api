# Windows.Media.SpeechRecognition.VoiceCommandSet.SetPhraseListAsync

## -description
Populates a [PhraseList](voice_command_elements_and_attributes.md) element with an array of [Item](voice_command_elements_and_attributes.md) elements.

## -parameters
### -param phraseListName
The string that corresponds to the label attribute of the [PhraseList](voice_command_elements_and_attributes.md) element.

### -param phraseList
A string array of values that will be added to the [PhraseList](voice_command_elements_and_attributes.md) element as [Item](voice_command_elements_and_attributes.md) elements.

## -returns
An asynchronous action.
