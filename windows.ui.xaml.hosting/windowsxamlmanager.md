# Windows.UI.Xaml.Hosting.WindowsXamlManager

## -description
Provides a static method you can use to initialize [Windows.UI.Xaml.UIElement](../windows.ui.xaml/uielement.md) objects for the current thread so they can be hosted in a [DesktopWindowXamlSource](desktopwindowxamlsource.md) object.
