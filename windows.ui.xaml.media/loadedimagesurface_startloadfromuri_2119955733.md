# Windows.UI.Xaml.Media.LoadedImageSurface.StartLoadFromUri

## -description
Loads an image from the specified URI with the desired maximum size.

## -params
### -param uri
The URI from which the image is loaded.

### -param desiredMaxSize
The desired maximum size of the image surface.

## -returns
An instance of **LoadedImageSurface** with the loaded image.
