# Windows.UI.Xaml.Media.XamlLight.RemoveTargetElement

## -description

## -description
Removes a UIElement from being a target of a compostion light.

## -params
### -param lightId
The identifier for the compostion light.

### -param element
The element that the compostion light targets.
