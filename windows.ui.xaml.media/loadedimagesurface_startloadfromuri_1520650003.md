# Windows.UI.Xaml.Media.LoadedImageSurface.StartLoadFromUri

## -description
Loads an image from the specified URI.

## -params
### -param uri
The URI from which the image is loaded.

## -returns
An instance of **LoadedImageSurface** with the loaded image.
