# Windows.Phone.Notification.Management.AccessoryManager.ClearToast

## -description
When a toast message is read on an accessory device, it can cause the toast to be cleared on the phone by using this method. The entry in the Action Center for the toast will be cleared as a visible result of calling this method.

## -parameters
### -param instanceId
String containing the instance id of the toast to clear.

## -remarks
The [VersionAvailable](http://msdn.microsoft.com/library/a9b27475-77d4-4e5c-8577-ca155232afdb) property can be used to determine which version of Windows Phone includes a specific API.
