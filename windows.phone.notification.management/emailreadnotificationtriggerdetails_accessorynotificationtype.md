# Windows.Phone.Notification.Management.EmailReadNotificationTriggerDetails.AccessoryNotificationType

## -description
An [AccessoryNotificationType](accessorynotificationtype.md) that specifies the type of notification received. For email notifications, this value is AccessoryNotificationType.EmailRead.

## -property-value
AccessoryNotificationType.EmailRead

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.

The [VersionAvailable](http://msdn.microsoft.com/library/a9b27475-77d4-4e5c-8577-ca155232afdb) property can be used to determine which version of Windows Phone includes a specific API.
