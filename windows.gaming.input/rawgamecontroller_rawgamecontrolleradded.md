# Windows.Gaming.Input.RawGameController.RawGameControllerAdded

## -description

Signals when a new raw game controller is connected.

## -remarks

To identify controllers that have already been added, you query the list of connected controllers using [RawGameController.RawGameControllers](rawgamecontroller_rawgamecontrollers.md).
