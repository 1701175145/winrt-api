# Windows.Gaming.Input.UINavigationController.GetRequiredButtonLabel

## -description
Retrieves a label for a required UI navigation button.

## -parameters
### -param button
Enumeration indicating which UI navigation button to retrieve the label for.

## -returns
The label of the specified button.
