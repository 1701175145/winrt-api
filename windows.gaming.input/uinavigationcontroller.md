Most Gamepads, ArcadeSticks, and RacingWheels can also be accessed as a UINavigationController. It is recommended that games use the controller specific class during gameplay and the UINavigationController class for UI menus.## -remarks## -examples## -see-also[IGameController](igamecontroller.md)
