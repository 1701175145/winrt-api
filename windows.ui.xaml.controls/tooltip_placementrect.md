# Windows.UI.Xaml.Controls.ToolTip.PlacementRect

## -description
Gets or sets the rectangle that the tool tip should be positioned in relation to when opened by the [ToolTipService](tooltipservice.md). If space allows, the open tool tip will not occlude the area defined by its PlacementRect.
