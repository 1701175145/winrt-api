# Windows.UI.Input.Spatial.SpatialTappedEventArgs.TapCount

## -description
The tap number represented by this gesture, either 1 or 2. A TapCount of 2 requires setting a SpatialGestureSettings value of DoubleTap.

## -property-value
The tap count.
