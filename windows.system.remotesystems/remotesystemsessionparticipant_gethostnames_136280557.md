# Windows.System.RemoteSystems.RemoteSystemSessionParticipant.GetHostNames

## -description
Retrieves a list of host names for the remote system that are resolvable by other remote systems in the session. TBD1

## -returns
A list of [**HostName**](https://docs.microsoft.com/en-us/uwp/api/windows.networking.hostname) objects for the host names of this remote system.
