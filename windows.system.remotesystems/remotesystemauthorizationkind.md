# Windows.System.RemoteSystems.RemoteSystemAuthorizationKind

## -description
Contains values specifying whether the client device can discover only same-user devices or other users' devices as well.

## -enum-fields
### -field SameUser:0
The client device can only discover devices signed in by the same user. TBD - what about hub devices?

### -field Anonymous:1
The client device can discover other users' devices. TBD - what limits this set?
