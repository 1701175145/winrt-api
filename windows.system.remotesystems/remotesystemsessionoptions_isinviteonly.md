# Windows.System.RemoteSystems.RemoteSystemSessionOptions.IsInviteOnly

## -description
Indicates whether or not the session described by this [**RemoteSystemSessionOptions**](RemoteSystemSessionOptions.md) instance should be joinable by invite only.

## -property-value
**True** if the corresponding session will only be available to participants who receive an invitation, **false** if the session will be open to join requests (or rather discoverable TBD1, how is this implemented?) from any participants that can discover it.
