# Windows.Graphics.Display.BrightnessOverride

## -description
Provides the ability to offer per-application brightness control on devices that support controllable brightness. This class allows apps to temporarily override the user's setting for brightness and automatically reverts to the brightness level as specified originally in the user's settings after the override session is over.
