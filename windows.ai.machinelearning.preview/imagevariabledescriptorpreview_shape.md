# Windows.AI.MachineLearning.Preview.ImageVariableDescriptorPreview.Shape

## -description
Gets the shape of the image variable.

## -property-value
The image shape, defined as (C, H, W) where C is the number of channels, H is the height of the image, and W is the width of the image.
