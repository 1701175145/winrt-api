# Windows.Management.Policies.NamedPolicy.TryCreatePolicySetterForUser

## -description
Attempts to create an MDM policy setter for the specified user object and account ID. 

## -parameters
### -param user
The user object of the policy setter.

### -param accountId
The account ID of the policy setter.

## -returns
A [NamedPolicySetter](namedpolicysetter.md) object for setting MDM policies.
