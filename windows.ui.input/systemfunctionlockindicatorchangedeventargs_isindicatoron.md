# Windows.UI.Input.SystemFunctionLockIndicatorChangedEventArgs.IsIndicatorOn

## -description

Gets the state of the Function key lock indicator for the [SystemFunctionLockIndicatorChanged](systembuttoneventcontroller_systemfunctionlockindicatorchanged.md) event.

## -property-value

True, if the indicator is on. Otherwise, false.
