# Windows.UI.Xaml.Media.Animation.ConnectedAnimation.IsScaleAnimationEnabled

## -description
Gets or sets a value that indicates whether scale animations can be used.

## -property-value
**true** if scale animations can be used; otherwise, **false**. The default is **true**.
