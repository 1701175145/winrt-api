# Windows.Graphics.Printing.OptionDetails.PrintTaskOptionDetails.CreateTextOption

## -description
Creates a [CustomPrintTaskOptionText](printcustomtextoptiondetails.md) object to handle the display name and other parameters of the advanced print task option item.

## -parameters
### -param optionId
The ID of the print task option.

### -param displayName
The display name of the print task option.

## -returns
The **CustomPrintTaskOptionText** object.
