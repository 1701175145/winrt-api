# Windows.UI.ViewManagement.ApplicationView.TemporaryInputMismatch

## -description

Gets or sets a value that indicates whether there is a temporary input mismatch.

## -property-value

**true** if there is a temporary input mismatch; otherwise, **false**. The default is **false**.
