# Windows.UI.ViewManagement.StatusBar

## -description
Provides methods and properties for interacting with the status bar assocaited with an app view (window). The status bar is a user experience that the system presents that expresses the state of an operation. It is most useful when the application view is not visible to the user.

## -remarks
To obtain an object of this type, call [StatusBar.GetForCurrentView](statusbar_getforcurrentview.md).
