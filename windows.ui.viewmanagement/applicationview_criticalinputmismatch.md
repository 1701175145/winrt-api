# Windows.UI.ViewManagement.ApplicationView.CriticalInputMismatch

## -description

Gets or sets a value that indicates whether there is a critical input mismatch.

## -property-value

**true** if there is a critical input mismatch; otherwise, **false**. The default is **false**.
