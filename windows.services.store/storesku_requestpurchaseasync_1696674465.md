This value corresponds to the [RPC_S_CALL_FAILED](https://msdn.microsoft.com/en-us/library/windows/desktop/aa378645(v=vs.85).aspx) error code.## -examples## -see-also[RequestPurchaseAsync(StorePurchaseProperties)](storesku_requestpurchaseasync_1703896342.md)
